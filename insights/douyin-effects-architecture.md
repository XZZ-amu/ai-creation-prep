# 抖音特效技术架构：端侧 vs 云端

> 跨 topic 的基础认知。所有 04-douyin-take.md 的产品判断都建立在这个架构理解之上。

---

## 一句话总结

**抖音现有特效 = 手机端跑小 AI + 实时渲染（成本为零）。你要加入的 AI 特效 = 云端跑大模型生成新画面（每次花钱）。两层并存，核心问题是"什么场景值得花云端大模型的钱"。**

---

## 两层架构

| | 第一层：端侧特效（已有） | 第二层：AI 生成特效（新增） |
|---|---|---|
| 跑在哪 | 用户手机 GPU | 字节云端 GPU 集群 |
| 引擎 | 火山引擎 Effect SDK（OpenGL ES / Metal） | Diffusion 推理平台（推测） |
| 延迟 | <37ms/帧（实时 30fps） | 2-60 秒（异步） |
| 边际成本 | 0 | 0.01-3 元/次 |
| 能做什么 | 变形已有画面（美颜、贴纸、滤镜、换背景） | 凭空创造新画面（AI 写真、风格重绘、文生图/视频） |
| 不能做什么 | 不能创造不存在的内容 | 不能实时（太慢） |
| 用户规模 | 全量用户日常使用 | 新功能，渗透率在爬坡 |

## 第一层：端侧 Effect SDK 能力清单

来源：火山引擎"智能美化特效"公开文档。

### AI/CV 能力（全部端侧推理）

| 能力 | 延迟 | 用途 |
|---|---|---|
| 人脸检测（106/280 点） | <2ms (iPhone 7) | 美颜、贴纸定位 |
| 表情驱动（52 blendshape + 3D 姿态） | <37ms (OPPO R11) | Avatar 驱动、表情特效 |
| SLAM（6-DOF） | 实时 | AR 物体放置 |
| 人像/头发分割 | 实时 | 换背景、背景虚化 |
| 骨骼/姿态检测 | 实时 | 全身特效、舞蹈特效 |
| 手势识别（静态+动态） | 实时 | 手势触发特效 |
| 宠物脸识别 | 实时 | 宠物贴纸 |
| 智能调色 | 实时 | 滤镜 |
| 超分/插帧 | 准实时 | 画质增强 |

### 渲染能力

- 支持同时追踪 10 张脸
- 输入：glTexture2D / RGBA / NV21 / YUV420P
- 推荐分辨率：720P-1080P
- 全部在 GL 线程完成
- iOS 支持 Metal（SDK v4.4.0+）

### TikTok Effect House（外部创作者平台）

- 渲染：WebGL
- 脚本：TypeScript / Visual Scripting
- 包体限制：8MB
- 加载时间限制：4 秒
- 图片限制：1024×1024px

## 第二层：AI 生成特效（你的团队）

### 与第一层的本质区别

| 端侧小 AI | 云端大 AI |
|---|---|
| 模型小（几 MB-几十 MB） | 模型大（几 GB-几十 GB） |
| 做"理解"（检测、分割、追踪） | 做"创造"（生成全新像素） |
| 输出：坐标/mask/系数 | 输出：完整图像/视频 |
| 确定性（同输入同输出） | 随机性（同 prompt 不同结果） |
| 不需要网络 | 必须联网 |

### 成本构成

模型开源 ≠ 免费使用。成本结构：

| 成本项 | 占比 | 说明 |
|---|---|---|
| GPU 硬件购买/折旧 | 60-70% | H100 约 25-35 万/张 |
| 电费 | 10-15% | 700W/张满载 |
| 机房/网络/散热 | 10-15% | 数据中心运营 |
| 运维人力 | 5-10% | 工程师团队 |

核心计量单位：**GPU·秒/次**。一张图 ≈ 2-10 GPU·秒，一段视频 ≈ 60-300 GPU·秒。

## 混合方案：两层之间的高价值地带

**核心思路：AI 做一次"理解/生成"（花一次钱），结果交给端侧渲染多次（免费）。**

### 示例

| 特效 | 纯 AI 方案（每次花钱） | 混合方案（花一次钱） |
|---|---|---|
| 换背景 | 每帧 AI 生成新背景 | AI 分割人像一次 → 端侧实时替换背景 |
| 风格滤镜 | AI 风格迁移每张图 | AI 生成风格 LUT/参数 → 端侧实时应用 |
| 动态贴纸 | AI 每帧生成 | AI 生成贴纸素材一次 → 端侧追踪贴合 |
| AI 写真 | 必须纯 AI | 无混合方案（必须生成新像素） |

### 判断原则

- 如果特效本质是"变换已有画面" → 优先混合方案
- 如果特效本质是"创造不存在的画面" → 必须纯 AI
- 中间地带：AI 生成关键素材（一次）+ 端侧组合渲染（多次）

---

## 对产品判断的影响

1. **不是所有"AI 特效"都需要云端大模型**——很多可以用端侧小 AI + 渲染搞定，成本为零
2. **真正需要云端大模型的场景要精挑**——只有"凭空创造"类才值得花这个钱
3. **混合方案是成本最优的创新空间**——一次 AI 推理支撑用户反复玩，ROI 最高
4. **端侧能力在持续增强**——今天必须云端做的，2-3 年后可能端侧就能做
5. **算力预算应该集中在"端侧绝对做不到"的事上**——比如高质量风格重绘、AI 写真、视频生成
