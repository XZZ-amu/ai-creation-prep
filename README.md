# AI 创作准备

围绕 AI 创作（生图、生视频）领域做问题驱动的深度研究，建立产品判断力。

## 结构

- `CLAUDE.md` — 项目协作规则与方法论
- `product-foundation.md` — 产品定位与贯穿性约束
- `topic-list.md` — 研究 topic 清单与状态
- `topics/` — 每个 topic 的研究单元（六步闭环）
- `insights/` — 跨 topic 的独立洞察
- `glossary.md` — 术语表

## 方法论

每个 topic 跑完整闭环：理解机制 → 识别问题 → 对比解法 → 抖音判断 → 动手实践 → 沉淀体感。

详见 `CLAUDE.md`。
